# Uso Diario

## Para la menor usuaria

Lo normal es usar WhatsApp con la wake word `studio`.

Ejemplos seguros:

- `studio abre blender`
- `studio crea proyecto castillo`
- `studio abre proyecto castillo`
- `studio haz una prueba de blender`
- `studio abre comfyui`
- `studio reinicia comfyui`
- `studio como esta comfyui`

## Si algo no responde

1. Abrir el acceso directo `OpenClaw Health`.
2. Si ComfyUI no abre, usar `Restart Studio Services`.
3. Si sigue fallando, avisar al adulto responsable.

## Lo que no debe hacerse

- no usar rutas absolutas raras en los proyectos
- no montar discos internos para “mirar algo”
- no tocar servicios desde terminal salvo mantenimiento adulto
